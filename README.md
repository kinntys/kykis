# kyki

kyki - это твое личное пространство, где пересекаются вдохновение, полезные технологии и живое общение.

## О проекте

kyki собирает все важное в одном окне и добавляет то, чего нет больше нигде - возможность общаться напрямую, даже когда интернет недоступен.

## Ключевые возможности

**Единый центр информации**
Выводи данные из любых приложений прямо на главный экран kyki. Погода, курсы валют, уведомления - все важное всегда перед глазами.

**Библиотека вдохновения**
Публикуй свои работы (фото, арты, заметки), сохраняй понравившееся от других пользователей и собирай свою уникальную коллекцию.

**Конструктор виджетов**
Создавай уникальные виджеты с любой информацией. Курс биткоина, список дел, заряд наушников - настраивай экран под себя.

**Bluetooth-чат**
Общайся с людьми рядом без доступа к интернету. Никаких серверов, только прямое соединение и полная приватность.

## Bluetooth-чат

kyki ломает цифровые барьеры. В приложении есть встроенный Bluetooth-чат, который работает без доступа к интернету.

- Полная приватность - никаких серверов и слежки
- Общение в метро, походах и местах с плохой связью
- Знакомства с людьми в радиусе действия Bluetooth
- Мгновенный обмен сообщениями и файлами

## Технические требования

- Android 8.0 и выше
- Bluetooth 4.0 и выше (для работы чата)
- 50 МБ свободного места
